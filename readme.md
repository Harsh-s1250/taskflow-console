# TaskFlow Console

A console-based task management system built using Java, JDBC, and MySQL.

## Features

- User Registration
- User Login
- SHA-256 Password Hashing
- Project Management
- Task Management
- Task Status Updates
- Dashboard Statistics
- Ownership-Based Access Control

## Technologies Used

- Java
- JDBC
- MySQL
- Git
- GitHub

## Database

Tables:
- users
- projects
- tasks

## How to Run

1. Clone repository
2. Create MySQL database
3. Update DB credentials in DBConnection.java
4. Run Main.java
